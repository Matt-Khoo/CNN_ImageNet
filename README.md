# CNN_ImageNet
Image classification using a Convolutional Neural Network (CNN) on ImageNet Animals dataset. 

## Instructions:
1. Install the following packages into your virtual environment (recommended) or local machine:
    - tensorFlow
    - keras
    - scikit-learn
    - pandas
    - matplotlib
    - opencv
    - imutils
    - cleverhans (install tensorflow first)
    
    Install the latest versions available for your venv or machine. <br>
    Installing cleverhans guide: https://github.com/tensorflow/cleverhans <br>
    Setting up Anaconda virtual environment: https://docs.conda.io/projects/conda/en/latest/user-guide/tasks/manage-environments.html <br>

2. Download the dataset (500 MB) from this link: https://drive.google.com/file/d/1wTFdkLHdfVbMBfLeKf68Bh1Hgvw8h_-D/view?usp=sharing
    
3. Extract "datasets.zip" into the same directory as this repository.

4. Run the cells in "CNN_ImageNet.ipynb" and observe the results.
